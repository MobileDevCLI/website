// Quick Start

Up and Running in Minutes

Three simple steps to start building from your phone.

1

Install Termux

Download Termux from F-Droid. It's a full Linux terminal for Android.

f-droid.org/packages/com.termux
2

Run Setup

One command installs Node.js, Git, and your AI CLI of choice.

curl -sL mobilecli.com/setup | bash
3

Start Building

Launch Claude Code (or any AI) and describe what you want to build.

claude --dangerously-skip-permissions
