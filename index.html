Quick Start

Get started in 4 steps

From zero to coding in under 5 minutes.

1

Install Termux

Download from F-Droid for the latest version with all features enabled.

2

Run Setup

Paste the curl command. It installs Node.js, Git, GitHub CLI, and Claude Code.

3

Login to GitHub

Run gh auth login to connect your GitHub account.

4

Start Coding

Type cc and start building with Claude in Boris Mode.
