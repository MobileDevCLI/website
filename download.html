📝 Installation

1

Download the APK

Click the download button above. Your browser may warn about APK files - this is normal for apps outside the Play Store.

2

Enable Unknown Sources

Go to Settings > Security > Install unknown apps and enable it for your browser or file manager.

3

Install & Open

Tap the downloaded APK to install. Open MobileCLI - the bootstrap will auto-install on first launch (~30 seconds).

4

Install AI CLIs

Once bootstrap completes, install your preferred AI coding assistant:

pkg install nodejs -y && npm i -g @anthropic-ai/claude-code
5

Start Coding

Launch Claude Code and authenticate:

claude

⚠️ Not on Google Play

MobileCLI uses targetSdk 28, which is required for executing binaries from app storage. Google Play requires targetSdk 33+, which breaks this functionality. This is the same reason Termux was removed from Google Play. Distribution is via GitHub only.
