Quick Start

Phone to dev machine in 5 minutes. Just commands, no fluff.

1

Get Termux

Download from F-Droid (not Play Store - that version is dead):

https://f-droid.org/packages/com.termux/

Open Termux, allow storage access:

termux-setup-storage
Why F-Droid?
The Play Store version of Termux hasn't been updated since 2020. F-Droid has the latest version with proper Android 12+ support. Full details
2

Install Packages

Run this in Termux:

pkg update -y && pkg install nodejs-lts git gh -y
What this installs
  • nodejs-lts - JavaScript runtime (required for Claude Code)
  • git - Version control
  • gh - GitHub CLI for auth and repo management
Full details
3

Install Claude Code

Install globally:

npm install -g @anthropic-ai/claude-code

Login with Anthropic account:

claude
First run
When you run claude the first time, it opens a browser for Anthropic login. Use your Google/GitHub account or create one. The CLI gets better rates than the API. Full details
4

Connect GitHub

Authenticate so Claude can push code:

gh auth login

Choose: GitHub.com > HTTPS > Login with browser

Git config
Set your identity:
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
Full details
5

Enable Boris Mode

Run Claude autonomously without permission prompts:

claude --dangerously-skip-permissions
What is Boris Mode?
Named after Boris Cherny (Claude Code creator), this flag lets Claude run without asking permission for file edits, shell commands, etc. Essential for autonomous overnight tasks. Full details
6

Keep It Running

Prevent Android from killing Termux:

termux-wake-lock

Use tmux for persistent sessions:

pkg install tmux -y
tmux new -s dev
Reconnecting
After closing Termux, reopen and run:
tmux attach -t dev
Your session (and Claude) will be exactly where you left it. Full details
!

You're Done

Start coding by describing what you want:

claude "Create a React app with auth and deploy to Vercel"

Claude will write code, run commands, push to GitHub, and deploy - all autonomously.

Building a Business?

Here's the exact stack we used to build MobileCLI from a phone:

1

Domain + Email

Buy domain on Vercel ($20/yr). Get free email forwarding. Or use Google Workspace ($6/mo) for custom email.

2

GitHub Organization

Create org at github.com/organizations/new. Free tier is fine. This holds all your repos.

3

Vercel Deployment

Connect GitHub repo to Vercel. Auto-deploys on every push. Free tier includes SSL, CDN, previews.

4

Supabase Backend

Free tier: Postgres DB, Auth (GitHub/Google OAuth), Storage. All you need for a SaaS.

5

Stripe Payments

Add subscriptions with Stripe. Test mode is free. 2.9% + $0.30 per transaction when live.

6

Social Presence

Create X (Twitter) and YouTube accounts. Same handle as your domain. Post updates, demos, builds.

The AI Does Most of It

You create the accounts (Google, GitHub, Vercel, Supabase, Stripe). Then Claude handles:

  • Writing all the code (HTML, CSS, JS, backend)
  • Setting up OAuth integrations
  • Database schema and migrations
  • Stripe webhook handlers
  • Git commits and pushes
  • CI/CD and deployment
