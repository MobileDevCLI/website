The Development Journey

These projects represent days of experimentation trying different approaches to bring AI development tools to Android. We explored:

  • Multiple architectures: v1, v2, v3, optionA, optionB approaches
  • Proot integration: Running Linux userspace on Android without root
  • Alpine Linux: Lightweight distro for minimal footprint
  • Terminal emulation: PTY handling, JNI native code
  • Symlink handling: Android filesystem quirks with EROFS
  • GitHub Actions: Automated builds for rapid iteration

While these apps don't fully work yet, they represent significant progress toward the goal of running Claude Code on any Android device. The journey continues!
